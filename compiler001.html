One Solution of IDEA Cache Problem

Detail

In a terminal, under a project root, after I typed "mvn clean install", the result was "build success". But the IDEA showed that the project was missing some dependencies.

Solution

Step 1.

In the IDEA, delete the target folder and the iml file under the project.

Step 2.

In the terminal, under the project, type "mvn clean install".

Step 3.

In the IDEA, chose the project root then right-click "Reload from Disk". Chose the pom file of the project then right-click "Maven" and "Reload project". (At this moment, you can see the color of the project folder name brighter.)

Step 4.

In the IDEA, chose the project folder then Build Project.
